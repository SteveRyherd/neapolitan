Neapolitan Domain Switcher

Keyboard Shortcuts

Keyboard shortcuts can significantly speed up your workflow when switching between environments. Chrome extensions can suggest shortcuts, but cannot automatically set them for security reasons.

To configure keyboard shortcuts for this extension:

  1. Type chrome://extensions/shortcuts in your browser's address bar
  2. Find "Neapolitan Domain Switcher" in the list
  3. Click the pencil icon next to any command to set or change its shortcut
  4. Press your desired key combination

Note: Chrome doesn't allow extensions to create direct links to chrome:// URLs for security reasons. You'll need to manually type the address in your browser.

Recommended Shortcut Schemes

Here are four shortcut schemes we recommend:

The Classic - 3 Stacked Scoops (Recommended)

Keys on left side in a straight line, rows relate to environment level

Shortcut Mnemonic Action
Alt+W W like Website Switch to Production
Alt+S S like Staging Switch to Staging
Alt+X X like Xcode Switch to Development

Neapolitan

Uses the same keys as copy, paste, save. Chocolate, Strawberry, Vanilla.

Shortcut Mnemonic Action
Alt+C C for Chocolate/Code Switch to Development
Alt+S S for Strawberry/Staging Switch to Staging
Alt+V V for Vanilla/Verified Switch to Production

Easy as 1, 2, 3

Number keys represent the natural development progression

Shortcut Mnemonic Action
Alt+1 1st environment Switch to Development
Alt+2 2nd environment Switch to Staging
Alt+3 3rd environment Switch to Production

Banana Split

Direct initials, but discouraged as keys are far apart

Shortcut Mnemonic Action
Alt+D D for Development Switch to Development
Alt+S S for Staging Switch to Staging
Alt+P P for Production Switch to Production
